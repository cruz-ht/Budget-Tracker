# Overview 

>The Budget Tractor is a simple, user-friendly, and interactive application that allows user to track their income, expenses and overall budget in real time!

# Features

Add Income and Expenses

- *Enter a description and amount.*

-  *Submissions are validated before being added.*

- *Totals update instantly after each entry*



Live Budget Summary

- *Automatically calculates total income, total expenses, and remaining budget.*

- *Updates in real time—no page refresh needed.*


# How it Works
1. *Enter a description and amount for either income or expenses.*

2. *Click Confirm.*

3. *View the updated totals in the summary section at the top of the page.*
